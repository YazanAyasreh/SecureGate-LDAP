## 📖 Overview
In the age of credential harvesting, a single password is a single point of failure. **SecureGate-LDAP** is a hardened authentication proxy that introduces a **Zero-Trust** architecture to legacy infrastructure. By requiring three independent factors—**Directory Credentials**, **Human Verification**, and **Time-based OTP**—it ensures that only authorized personnel gain access.



---

## 🛠️ System Architecture

### 🛡️ The Triple-Lock Security Flow
1. **The Identity Lock:** Validates username/password against **Active Directory** or **LDAP3**.
2. **The Integrity Lock:** Challenges the request with **Google reCAPTCHA v2** to neutralize automated bot attacks.
3. **The Ownership Lock:** Verifies a 6-digit **TOTP** token generated by the user's mobile device (Google Authenticator/Authy).

---

## ✨ Features & Design
The UI is built with a **macOS-inspired minimalist aesthetic**, focusing on professional "SaaS" clarity:
* **Adaptive Security:** Automated account lockout after 5 failed attempts.
* **Cryptographic Safety:** Passwords hashed with `Bcrypt` (Salted & Peppered).
* **Hardened Headers:** Built-in protection against XSS, Clickjacking, and MiTM attacks.
* **Modern UI:** Clean, centered 400px containers with high-contrast typography.

## 🔐 Environment Configuration (.env)

The application requires a `.env` file in the root directory to manage sensitive credentials. **Never commit your actual `.env` file to version control.** Create a file named `.env` and populate it with the following:

```env
# Flask Security
SECRET_KEY=generate_a_secure_hex_string_here

# Google reCAPTCHA v2 (Get keys at: [google.com/recaptcha/admin](https://google.com/recaptcha/admin))
RECAPTCHA_PUBLIC_KEY=your_site_key_here
RECAPTCHA_PRIVATE_KEY=your_secret_key_here

# Database Configuration
MONGO_URI=mongodb://localhost:27017/
MONGO_DB_NAME=mfa_app

# LDAP / Active Directory Configuration
LDAP_SERVER=ldap://127.0.0.1
LDAP_BASE_DN=dc=example,dc=com
